¿Qué son las Distribuciones GNU-Linux?

Una Distribucion como ya comente en Linux, una Distribucion es un sistema oprativo que usa el motor de GNU-Linux, gracias a que el KernelLinux es Open Source, grupos de derolladores o empresas usan este motor para crear sus propios sistemas operativos, tal vez te suene el nombre Ubuntu, esta es una Distribucion GNU-Linux, pero es una de las más famosas. A la derecha te dejare un diagrama de los sistemas operativos GNU-Linux mas usados.

¿Cómo se hace estos sistemas oprativos?

Los sistemas operativos de este estilo son más faciles de hacer que los normales, poque todo el Kernel es ya fue escrito por otra persona, ósea , Linus (El creador de Linux) hizo el kernel y otras personas empezarón a añadirle aplicaciones al punto que poco a poco se empezarón hacer diferentes sistemas oprativos, por ejemplo, uno de las distribuciones más antiguas es Debian, esta creo sistemas de descarga, creo sus propias aplicaciones, añadio la interfaz grafica GNOME, si no es suficiente eso, esto no lo hizo una empresa sino un grupo de desarolladores y cada uno se encargo de una parte de Debian. Y Ubuntu se basa en esta distribucion al ser Open Source como la mayoria de estas.
