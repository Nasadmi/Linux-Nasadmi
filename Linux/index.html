¿Qué es Linux?

Linux es un 'Sistema Operativo', esta entre comillas porque si hablamos de Linux como tal no es un sistema operativo, más bien es el Kernel, pero si se le une un grupo de aplicaciones si se hace un sistema operativo. Linux es un núcleo el cual permite hacer diferentes sistemas operativos y a diferencia de MacOS o Windows es "Open Source" o en español "Codigo Abierto" esto significa que cualquier persona puede ver, modificar y publicar el código como si fuese suyo sin necesidad de pedrile permiso al autor, con linux se puede hacer lo que se conoce como Distribuciones que son sistemas operativos que la mayoría es de software libre los cuales usan como base Linux.

¿Por qué no es tan conocido?

A pesar de que Linux no es tan conocido no significa que no se use, Linux en el hambito de Escritorio no es conocido pero en el hambito de servidores es el sistema operativo más usado, esto se debe a dos razones. La primera es que casi todas las distribuciones Linux son gratuitas y mucho mas ligeras que Windows, y la segunda es que un servidor solo mueve datos así para que voy a instlarle Windows si solo mueve datos. El problema es que Linux no es usado en computadoras de escritorio ya que la creencia popular dice que es un sistema opertaivo díficil de usar. Pero depende de la distribucion que uses, porque hay distribuciones más complicadas de usar que otras.
